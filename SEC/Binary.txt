Binary to decimal

We start from right with 2^0

0000 1100

0 0 0 0 1 1 0 0 == 12

0*2^7 + 0*2^6 + 0*2^5 + 0*2^4 + 1*2^3 + 1*2^2 + 0*2^1 + 0*2^0 == 12
Decimal to binary

We check a modulo % 2. When result is 0 we stop.

12 % 2 = 6 = 0

6 % 2 = 3 = 0

3 % 2 = 1 = 1

1 % 2 = 0 = 1

Then we read from the bottom 1100 == 0000 1100 == 12
